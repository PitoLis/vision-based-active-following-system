# vision-based-active-following-system
基于机器视觉的主动跟随系统，实现目标检测、跟踪与自主移动控制。

## Part1-moving parts
based on STM32 realizes functions such as line tracking, Bluetooth, WiFi and ultrasonic obstacle avoidance.
## Part2-tracking parts
Two-Dimensional Pan-Tilt Platform Based on Stepper Motor
